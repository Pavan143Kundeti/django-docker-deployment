About This Project

This is a demonstration Django web application designed to teach Docker containerization concepts.

🎯 Project Purpose

This project demonstrates how to containerize a Django application using Docker, solving the "it works on my machine" problem by packaging the application with all its dependencies.

🛠️ Technology Stack

  • Framework: Django 4.2.0
  • Language: Python 3
  • Container: Docker
  • Base Image: Ubuntu

📦 What's Included

  • Complete Django project structure
  • Dockerfile for containerization
  • Requirements.txt for dependencies
  • Demo application with views and templates
  • Ready-to-deploy configuration

🚀 How to Run

1. Build the Docker image:
docker build -t django-app .

2. Run the container:
docker run -p 8000:8000 django-app

3. Access in browser:
http://localhost:8000
